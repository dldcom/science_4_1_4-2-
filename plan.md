# 🌌 우주 미생물 머지 타이쿤 (Cosmic Microbe Merge) 기획 및 개발 계획

우주를 배경으로 다양한 미생물(원생생물, 균류, 세균)을 퀴즈로 소환하고, 이들을 조합 연구소에서 합성하여 새로운 변종을 만들어 우주 자원을 채굴하는 교육용 타이쿤 게임입니다.

---

## 1. 게임 콘셉트
- **배경:** 신비로운 네온 컬러의 우주 성운
- **장르:** 퀴즈 & 합성 타이쿤 (Merge Tycoon)
- **주요 기능:**
  1. **과학 퀴즈 풀기:** 정답을 맞히면 미생물이 우주 공간에 탄생 (소환 연출)
  2. **생명 합성:** 2~3마리의 미생물을 조합법에 맞게 합성하여 상위 변종 획득
  3. **자동 채굴:** 미생물들이 우주 성운을 공전하며 자원을 자동 수확 및 수집기에 배달
  4. **영토 확장:** 모은 우주 자원으로 구역(Sector)을 해금하여 수용량과 신규 광산 확보

---

## 2. 미생물 정의 및 특징
교과서에 나오는 핵심 생물들의 시각적 특징과 이동 방식을 고스란히 반영합니다.

| 종류 | 이름 | 생김새 특징 | 이동/행동 특징 |
| :--- | :--- | :--- | :--- |
| **원생생물** | 짚신벌레 | 짚신 모양, 전신 섬모 | 섬모가 물결치듯 파르르 떨리며 고속 헤엄 |
| | 아메바 | 정해진 모양 없음, 반투명 | 위족(거짓발)을 뻗으며 느릿느릿 기어감 |
| | 해캄 | 나선형 엽록체, 대나무 마디 | 움직이지 않고 햇빛을 받아 광합성(마디 성장) |
| **균류** | 버섯 | 갓과 자루 형태 | 보이지 않는 균사로 자원을 먹고 싹트는 연출 |
| | 곰팡이 | 거미줄 균사, 끝의 포자낭 | 한곳에 솜털처럼 퍼지며 포자를 흩뿌려 번식 |
| **세균** | 대장균 | 막대 모양(간균), 편모(꼬리) | 꼬리를 돌리며 부르르 진동하듯 이동 |
| | 젖산균 | 동글동글한 구균 | 세포가 뚝 잘라지는 이분법 분열 |

## 3. 생명 합성 (조합) 레시피
조합소 UI에서 미생물들을 조합하여 우주 자원 채굴 능력이 월등히 강화된 변종 미생물을 탄생시킵니다.

- **[버섯] + [버섯] = [화산 버섯]** (우주 열기를 흡수, 채굴 속도 2배)
- **[곰팡이] + [곰팡이] = [네온 곰팡이]** (주변 미생물의 자원 채굴 효율 30% 증가 버프)
- **[버섯] + [곰팡이] = [포자버섯 젤리]** (성운 주변 자원을 넓은 반경에서 자동 흡입)
- **[짚신벌레] + [짚신벌레] = [제트 짚신벌레]** (성운 주변을 매우 빠른 속도로 공전하며 자원 신속 수집)
- **[아메바] + [아메바] = [블랙홀 아메바]** (한 번에 많은 양의 자원을 채굴하여 수집기에 즉시 배달)
- **[유글레나] + [아메바] = [태양 아메바]** (최상위 변종, 스스로 에너지를 대량 생산)

---

## 4. 자원 수확 및 영토 확장
- **자동 채굴:** 소환된 미생물들이 성운 주위를 맴돌며 '우주 바이오 에너지(Bio-Energy)'를 수집합니다.
- **수집기 배달:** 미생물의 자원 가방이 가득 차면(100%), 화면 중앙의 '차원 에너지 수집기'로 신속히 날아가 자원을 비우고 복귀합니다.
- **영토 확장:** 모은 바이오 에너지로 장막에 가려진 다음 섹터(Sector)를 해금하여 최대 소환 한도를 늘리고 더 거대한 성운을 확보할 수 있습니다.
